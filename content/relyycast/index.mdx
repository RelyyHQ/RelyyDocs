# RelyyCast Overview

RelyyCast helps you turn a source audio feed into shareable stream links with a guided desktop control panel.

## Who this is for

Creators and broadcasters who want a reliable stream workflow without deep infrastructure setup.

## What you can do with RelyyCast

- Configure your station metadata and ingest settings.
- Start a local MP3 and HLS output.
- Enable Cloudflare for a public URL with either a temporary link or custom domain.
- Check runtime, relay, and listener health from one control screen.

## Supported platforms

- macOS
- Windows
- Linux support is planned, but not part of this first user-docs release.
