# Download and Install

## Who this is for

Anyone installing RelyyCast for the first time.

## Prerequisites

- macOS or Windows.
- Admin permissions for software installs.

## Steps

1. Download the latest RelyyCast release from GitHub Releases: [RelyyCast Releases](https://github.com/relyy/relyycast/releases).
2. Install and launch RelyyCast.
3. Install required dependencies:
   - FFmpeg: [ffmpeg.org/download.html](https://ffmpeg.org/download.html)
   - MediaMTX: [github.com/bluenviron/mediamtx/releases](https://github.com/bluenviron/mediamtx/releases)
   - cloudflared (optional for public URLs): [Cloudflare Downloads](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/)
4. Reopen RelyyCast after dependency install.

## What success looks like

- The app launches without errors.
- You can open the `Settings` tab and save configuration.

## Common mistakes

- Installing only the app but not FFmpeg and MediaMTX.
- Forgetting that cloudflared is optional unless you need a public URL.
- Downloading from third-party mirrors instead of official release pages.
