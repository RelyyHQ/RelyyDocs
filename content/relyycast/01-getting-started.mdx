# Getting Started

## Who this is for

First-time creators who want to go from install to first successful stream with minimal setup overhead.

## Prerequisites

- A macOS or Windows machine.
- Permission to install apps and tools on your machine.
- A stable internet connection.

## Steps

1. Read [Download and Install](/relyycast/02-download-and-install) and install the app plus required dependencies.
2. Open RelyyCast and confirm you can access both `Control` and `Settings` tabs.
3. Complete your first station configuration in [First Stream Setup](/relyycast/03-first-stream-setup).
4. Use [UI Tour](/relyycast/04-ui-tour) to understand what each status means.

## What success looks like

- The app opens successfully.
- You can save settings.
- Runtime reaches `RUNNING`.
- You can open at least one working stream URL.

## Common mistakes

- Skipping dependency install before first launch.
- Expecting Cloudflare to connect automatically before clicking the connect action.
- Testing stream URLs before runtime finishes startup.
